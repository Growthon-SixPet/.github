# 🐾 With Tail <br> 반려동물의 시작부터 끝까지 함께하겠습니다.

## 📌 프로젝트 개요
With Tail은 반려동물을 위한 병원과 장례식장을 추천해주고, 예약까지 도와주는 웹사이트입니다.

## 👥 팀 구성
DE : 김규빈 <br>
FE : 서강호, 오민석 <br>
BE : 김한민, 박연지, 박유미

## ⚠️ 문제 정의와 서비스 목표
**문제 정의**
- 반려동물 장례업 관련 정보가 여러 플랫폼에 분산되어 있어 정보 파악 어려움.
- 동물병원과 장례까지의 연계 서비스 찾기 힘듬.
<br>

**서비스 목표**
- 진료와 장례를 통합한 서비스
- 필터를 통한 비교와 검색
- 접근성 좋고 정보 습득이 쉬운 서비스
<br>

최근 5년동안 반려동물을 키우는 사람들이 늘어나고 있습니다. <br>
이에 따라 국내에 반려동물 시장의 규모 또한 빠르게 성장하고 있고, 향후에도 반려동물과 관련된 서비스에 대한 수요는 지속적으로 확대될 것으로 예상됩니다. <br>
하지만 반려동물 장례에 대한 정보는 여러 곳에 파편화되어 있어 한눈에 파악하기 어렵고, 동물병원에 이어 장례식까지 통합된 서비스를 찾기 힘듭니다. <br>
<br>
따라서 진료와 장례까지 통합된 서비스, 한눈에 파악하기 쉬운 비교와 검색이 가능한 서비스, 정보 습득이 용이한 서비스를 제공하고자 합니다.

## ⭐ 주요 기능
- 회원가입, 로그인 기능
- 동물병원 / 장례식장 예약 기능
- 동물병원 / 장례식장 자체 후기 기능
- 마이페이지 (회원 정보, 예약 내역, 후기 내역, 즐겨찾기)
